Desktop Direction

GoodNight 适合做成“原生桌面感 + 品牌气质”的工作台

我建议保留你现在的四段式结构，不改骨架，只统一窗口层级、标题栏语言、面板质感和 AI 侧栏存在感。重点不是加更多装饰，而是把无边框窗口做成这个产品最有辨识度的部分。

Recommended

推荐优先做第三个方向：比纯工具感更有记忆点，比 IDE 风更克制。它最适合你现在这类“知识 / 设计 / 开发 / Agent”混合工作台。

Quiet Native 最克制

更接近 Linear / Raycast 那种安静、专业、原生感强的工具。标题栏尽量轻，主工作区最显眼。

  • 视觉负担最小，稳定耐看
  • 系统窗口手感最自然
  • 品牌记忆点相对弱一点

Studio Dense 操作感强

更像 IDE / 创作工作台，工具感更重，信息更满，适合强调多面板、多状态、多任务并行。

  • 专业工作台气质明确
  • 适合复杂流程和高密度状态
  • 稍不控制就会显得硬和乱

Signature Workbench 推荐

保留专业工作台的秩序感，但给标题栏、窗口框体和 AI 侧栏更完整的产品识别。更像“这是 GoodNight 自己的桌面工具”。

  • 原生感、品牌感、工作感更平衡
  • 最适合无边框窗口做出识别度
  • 后续也最容易同时做流畅度优化

GoodNight UI Standard Set

The reusable standard library for future frontend work. Organized more like a desktop source list and document browser than a marketing-style board.

4 core files Light + dark AI-readable
Foundation Standard ui-standards.html

Tokens, icon library, layout anatomy, workspace primitives, AI contracts, and implementation rules.
Desktop Component Standards desktop-component-standards.html

Ant Design-like component standard page for shell, navigation, work surfaces, states, AI lanes, templates, and React usage contracts.
Design Boards overview-home.html

Workbench boards for AI note flow, graph workspace, overview launcher, and lane rendering examples.
State Standard Sheet state-standards.html

Dedicated full-page visuals for default, hover, selected, collapsed, empty, loading, and error states.
